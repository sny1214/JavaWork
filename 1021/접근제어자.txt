1. 접근제어자는 클래스나 클래스의 멤버에 적용하는 제어자
   1) 클래스에 적용하는 접근 제어자
      default : 현재 패키지에서 마음대로 사용 할 수 있는 클래스 이다.
      public : 현재 패키지뿐만아니라 다른 패키지에서도 사용할 수 있는 클래스 이다.
      
   2) 멤버에 적용할수 있는 접근 제어자
      동일패키지            다른패키지
      ------------------ ------------------
      private < default < protected < public		
   	  내부 숨김    공개	      상속된클래스의   외부공개
   		                  메서드내부에서 사용가능
   		
      1. private - 같은 패키지이건 다른 패키지이건 외부에서 접근할 수 없다.
      2. default - 아무것도 사용하지 않은것
                    같은 패키지에서는 내.외부에서 자유롭게 사용
                    다른 패키지에서는 외부에서 접근할 수 없다.
      3. protected - 상속된 하위클래스의 메서드 내부 에서만 패키지가 달라도 자유롭게 접근
      4. public - 같은 패키지,다른 패키지 모두에서 외부에 자유롭게 접근
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

       3) 생성자를 대신할 public 메서드를 만든다(getInstance())
