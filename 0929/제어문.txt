1. 세가지 제어구조
   1) 순차구조 : 명령어를 위에서 아래로 차례로 수행하는 구조
   2) 선택구조 : 특정 조건에 따라 수행할 명령어를 선택하는 구조
      1) 단순 선택 구조 : 특정기능을 수행할지 말지만 결정
      2) 양자 택일 구조 : 참일때 수행할 기능과 거짓일때 수행할 기능이 따로 있을때 
      3) 다중 택일 구조 : 여러개에서 하나를 선택하여 실행
   3) 반복구조 : 특정 조건을 만족하는 동한 반복하는 구조

       
2. 제어문(statement)

   선택문
   1) 단순 if문
      if(조건){
         조건이 참일때 수행할 문장
      } 
      참고) 조건이 참일때 수행할 문장이 1개라면 블럭기호 즉, {} 생략할 수 있다.
   
   2) if else문
      if(조건){
         조건이 참일때 수행할 문장
      }else{
         조건이 거짓일때 수행할 문장   
      }
   
   3) else if문   
      if(조건1){
         실행문장1;
      }else if(조건2){
         실행문장2;
      }else if(조건3){
         실행문장3;
      }....
      else if(조건n){
         실행문장n;
      }else{
         모든 조건이 거짓일때 수행하는 문장;
      } 
   
   4) switch(변수){
      case 값1 : 수행문장1; break;
      case 값2 : 수행문장2; break;
      case 값3 : 수행문장3; break;
      ...
      case 값n : 수행문장n; break;
      defalut : 수행문장
      }
      
      
  반복문
  1) while 반복 ==> 반복제어변수의 증감이 일정하지 않다(불규칙 반복)
  
     반복제어변수초기화;
     while(반복조건){
        수행할 문장;
        반복제어변수증감;
     }
     
  2) for 반복  ==> 반복제어변수의 증감이 일정할때(규칙적인 반복)
     
     for(반복제어변수초기화;반복조건;반복제어변수증감){
         수행할 문장;
     } 
     
  3) do while반복
     do{
        수행할 문장;
     }while(반복조건);
