
1. 제어문(control statement) : 블럭구조
                             {
                               실행문장;
                             }
   참고) 실행문장 1개일때는 블럭구조를 생략할 수 있다.                           
                                 
 1) 선택문 : if문, if else문,  else if문, swith case문
 2) 반복문 : while문, for문, do while문

2. 분기문(brench statement) : 블럭구조 없이 하나의 명령어로 
   다음 실행위치를 결정하는 명령
   1) break; (1)반복문에서는 가장 가까운 반복구조를 강제탈출 시킨다.
             (2)switch문에서는 switch문을 탈출시킨다.
             
   2) continue; 반복문에서 반복조건으로 제어를 이동시킨다.             
   
   3) return [값];   메서드 실행을 무조건 종료하고 메서드를 호출하는 곳으로 되돌아 간다.
